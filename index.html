Automações com Bookmarklets
Scripts prontos que rodam direto no navegador. Arrasta uma vez para os favoritos e depois é só clicar — sem instalar nada.
O que é um bookmarklet?
Um favorito que executa código
É um favorito normal do navegador, mas em vez de guardar uma URL, ele executa um script JavaScript na página aberta. Roda localmente no seu navegador, sem enviar dados pra fora, sem instalar extensão.

Como usar — passo a passo
Parte 1 · Instalar (só uma vez)
1
Exiba a barra de favoritos
Pressione Ctrl+Shift+B para mostrar a barra de favoritos, caso ela esteja oculta.
2
Arraste o botão roxo para os favoritos
Segure o botão Intervalo 6 meses acima e arraste até a barra de favoritos do navegador. Solte quando aparecer o indicador de posição. Feito — está instalado!
Parte 2 · Usar no dia a dia
1
Abra o Digiteam na aba Despacho
Navegue até a tela de Despacho normalmente. O script precisa que a página esteja carregada.
2
Selecione seu filtro de equipe / setor
Aplique o filtro que você usa normalmente (equipe, regional, tipo de serviço…) antes de acionar o bookmarklet.
3
Clique no favorito "Intervalo 6 meses"
Clique no favorito instalado na barra. O script roda sozinho — abre o filtro, ativa tempo real, preenche as datas (6 meses) e configura atualização de 1 min. Aguarde uns 3 segundos e tá pronto.
Dicas e observações
O bookmarklet funciona somente no Digiteam. Em outras páginas ele não vai encontrar os elementos e pode exibir um alerta.
Se a tela do Digiteam mudar de layout em uma atualização futura, o script pode parar de funcionar. Nesse caso, chama o Lucas.
Você pode instalar o bookmarklet em mais de um navegador. Basta repetir o processo de arrastar em cada um.

Renomeador de Equipes
Dois bookmarklets: o primeiro carrega um CSV com os dados das equipes, o segundo percorre o Digiteam renomeando tudo em lote — pulando bloqueadas e já atualizadas.
Como funciona
1️⃣
Preparar o CSV

→
2️⃣
Bookmarklet: Conectar CSV
Abre seletor de arquivo e salva os dados no navegador.
→
3️⃣
Bookmarklet: Renomear
Percorre o Digiteam e renomeia tudo em lote.
Passo 1 — Preparar o CSV
Crie um arquivo .csv separado por vírgulas com exatamente três colunas nessa ordem:
Coluna A
veiculo
Número do veículo (4 dígitos)
Coluna B
localdecadastro
Local / bairro / regional
Coluna C
termino
Horário de término (ex: 17H)
Exemplo de arquivo CSV
veiculo,localdecadastro,termino
1023,BOM DESPACHO,17H
1047,DIVINOPOLIS,16H
1088,PARA DE MINAS,17H
1102,ITAUNA,18H
💡 O script vai gerar o nome no formato LOCAL-VEICULO-TERMINO, ex: BOM DESPACHO-1023-17H. Acentos são removidos automaticamente.
Passo 2 — Conectar CSV
Abre o arquivo e salva os dados no navegador para o Renomeador usar

Tutorial — do zero ao renomeado
Preparação (uma vez)
1
Monte o CSV com os dados das equipes
Três colunas: veiculo, localdecadastro, termino. Salve como .csv.
2
Instale os dois bookmarklets
Arraste Conectar CSV e Renomear Equipes para a barra de favoritos (Ctrl+Shift+B para exibir).
No dia de renomear
3
Abra o Digiteam e clique em Conectar CSV
Selecione o arquivo .csv. Uma confirmação vai aparecer com o total de veículos carregados.
4
Clique em Renomear Equipes — aguarde
Uma barra de progresso vai aparecer no canto da tela. Você pode pausar ou parar a qualquer momento. Ao final, um resumo mostra sucesso, falhas e equipes bloqueadas.
Dicas e comportamentos
O CSV precisa ser carregado a cada nova sessão — os dados não sobrevivem se você fechar e reabrir a aba.
Equipes com o nome já correto são detectadas e ignoradas — seguro de rodar mais de uma vez.
Equipes com 🔒 cadeado são puladas automaticamente e contadas separado no resumo final.
O formato do nome gerado é sempre LOCAL-VEICULO-TERMINO em maiúsculas sem acentos, ex: PARA DE MINAS-1088-17H.
